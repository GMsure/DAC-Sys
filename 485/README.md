

硬件资源:
	1,DS0(连接在PB5)
	2,串口1(波特率:115200,PA9/PA10)
	4,按键KEY0(PE4) 
	3,串口2(波特率:9600,连接在PA2/PA3)  
	3,P5跳线帽的连接状态必须是:48T连接PA3,48R连接PA2.
	4,RS485接口的A/B线不能接错,必须A接A,B接B.

| ----------------- | ------ | --------------------------------- |
| PA9               | 杜邦线 | ESP8266 / RX                      |
| PA10              | 杜邦线 | ESP8266 / TX                      |
| PA2               | 跳线帽 | 板载 / 48R                        |
| PA3               | 跳线帽 | 板载 / 48T                        |
| RS485A            | 杜邦线 | 485总线(三个传感器的) / A         |
| RS485B            | 杜邦线 | 485总线(三个传感器的) / B         |
| 5V                | 杜邦线 | ESP8266 / VIN                     |
| GND               | 杜邦线 | ESP8266 / GND                     |
| GND               | 杜邦线 | 485总线(三个传感器的) / GND(12V-) |
